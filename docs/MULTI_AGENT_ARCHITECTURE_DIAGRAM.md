## 🔄 二、消息流转图

### 2.1 QQ Bot 消息流转

```
用户 (QQ) ──→ QQ Bot 通道 ──→ qqbot agent (细佬)
                                    │
                                    ├──→ 系统问题？──→ 自己处理
                                    │
                                    ├──→ 投资问题？──→ sessions_send → telebot
                                    │
                                    ├──→ 文档问题？──→ sessions_send → feishu
                                    │
                                    └──→ 紧急警报？──→ sessions_send → dashboard
```

### 2.2 Telegram 消息流转

```
用户 (Telegram) ──→ Telegram 通道 ──→ telebot agent (叻仔)
                                            │
                                            ├──→ 加密货币查询？──→ CoinGecko API
                                            │
                                            ├──→ 市场分析？──→ Tavily API
                                            │
                                            ├──→ 套利机会？──→ Polymarket API
                                            │
                                            └──→ 通知用户？──→ message (Telegram)
```

### 2.3 Feishu 消息流转

