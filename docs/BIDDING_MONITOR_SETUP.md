## 🔧 配置步骤（3 步搞定）

### 第 1 步：创建飞书机器人

1. **打开飞书** → 找到你想接收通知的群聊（或者创建私人对话）

2. **添加机器人**：
   - 点击右上角「...」或「设置」
   - 选择「添加机器人」
   - 选择「自定义机器人」

3. **配置机器人**：
   - 名称：`投标监控助手`
   - 头像：随便选
   - ✅ 勾选「发送消息」
   - ✅ 勾选「获取机器人所在群信息」

4. **复制 Webhook URL**：
   - 格式：`https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx`
   - 点击「复制」按钮

---

### 第 2 步：配置脚本

打开脚本文件：
```bash
nano /root/.openclaw/workspace/scripts/bidding_monitor.py
```

找到第 19 行，填入你的 Webhook：
```python
FEISHU_WEBHOOK = "https://open.feishu.cn/open-apis/bot/v2/hook/你的 webhook 地址"
```

保存退出（Ctrl+O → Enter → Ctrl+X）

---

### 第 3 步：设置定时任务（每日 10 点）

```bash
# 编辑 crontab
crontab -e

# 添加以下行（每日 10:00 执行）
0 10 * * * cd /root/.openclaw/workspace && python3 scripts/bidding_monitor.py >> data/bidding_monitor/cron.log 2>&1
```

**验证 cron 任务**：
```bash
crontab -l  # 查看已配置的任务
```
