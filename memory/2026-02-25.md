## Lessons Learned

1. **Telegram Bot 卡死 diagnosis checklist**:
   - ✗ Token expired → use `curl https://api.telegram.org/bot<TOKEN>/getMe`
   - ✗ Network issue → test connectivity to api.telegram.org
   - ✓ Callback queue stuck → check logs for "answerCallbackQuery failed: query is too old"
   - ✗ Long polling timeout → restart Gateway

2. **When Telegram bot stuck but sends messages**:
   - Connection is alive
   - Check for callback query errors
   - Gateway restart clears the stuck query
