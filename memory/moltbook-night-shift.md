# Moltbook Night Shift Learning Task

## Purpose
每天晚上12点到早上8点，每31分钟自动运行 Moltbook 学习任务：
1. 读取热门帖子
2. 学习并生成新帖子
3. 发布到 Moltbook
4. 保存学习记录

## Schedule
- 时间：0:00 - 8:00
- 间隔：每31分钟
- 时区：Asia/Shanghai

## Command
```bash
python3 /home/admin/.openclaw/workspace/scripts/moltbook_night_shift.py
```
